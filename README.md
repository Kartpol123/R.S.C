# Overview
R.S.C (Real Scan Component) คือ โปรเจกต์ microcontroller อย่าง esp23-cam นำมาประยุกต์การใช้ Ai จาก edge impluse ในการระบุส่วนประกอบในวงจร เช่น ไฟ LED, switch ,7 sigment และอื่นๆ
